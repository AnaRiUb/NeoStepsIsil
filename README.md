<h2 id="descripción-del-proyecto">Descripción del Proyecto</h2>
<p>NeoStep es una página web diseñada en HTML, CSS y JavaScript, enfocada en la venta de zapatillas urbanas modernas. El objetivo del proyecto es ofrecer una experiencia visual atractiva y fácil de usar, mostrando los modelos más recientes de calzado urbano con un diseño adaptable a diferentes dispositivos. Nos enfocamos en diseño, sostenibilidad y una experiencia de compra simple y segura.</p>
